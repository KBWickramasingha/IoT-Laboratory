# 📡 ESP32 MQTT Communication System

## 🎯 Objective
Implement MQTT-based IoT communication using ESP32.

---

## 📌 Features
- Publish sensor data
- Subscribe to topics
- Real-time communication

---

## 🧠 Learning Outcomes
- MQTT protocol
- Broker-based communication
- IoT messaging systems

---

## 🛠 Tools
- ESP32
- Mosquitto MQTT Broker
- Node-RED (optional dashboard)

---

## 🔗 Flow
ESP32 → MQTT Broker → Node-RED / Client
