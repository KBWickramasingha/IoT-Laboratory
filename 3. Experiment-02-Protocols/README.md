# 🌐 Experiment 02 – IoT Communication Protocols

## 🎯 Objective
To study and implement IoT communication methods using ESP32.

---

## 📡 Protocols Covered

### 1. HTTP
- ESP32 Web Server
- Control devices via browser

### 2. MQTT
- Publish/Subscribe model
- Real-time IoT communication

### 3. WebSocket
- Full-duplex communication
- Real-time data streaming

---

## 🧠 Learning Outcomes
- IoT communication architecture
- Client-server model
- Real-time data exchange systems

---

## 🛠 Tools
- ESP32
- Arduino / PlatformIO
- Node-RED
- MQTT Broker (Mosquitto)

---

## 🔗 Subfolders
- `/http` → HTTP Web Server project
- `/mqtt` → MQTT IoT system
- `/websocket` → real-time communication system
