# 🌐 ESP32 HTTP Web Server

## 🎯 Objective
Create a web server using ESP32 to control devices via browser.

---

## 📌 Features
- WiFi connection
- Web-based LED control
- Simple REST-style interface

---

## 🧠 Learning Outcomes
- HTTP protocol basics
- Embedded web server design
- WiFi communication

---

## 🛠 Hardware
- ESP32
- LED
- Resistor

---

## 🚀 Working Principle
Browser → HTTP Request → ESP32 → GPIO Control

---

## 🔮 Future Upgrade
- Add sensor data display
